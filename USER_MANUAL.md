# Fairway Command — Golf League Tracker Pro
## User Manual (Admin / Scorer / Player)

**Website URL:** https://sweet-dasik-f61f90.netlify.app

---

## 1) Install as an App (Recommended)

### Windows (Chrome or Edge)
1. Open the website URL.
2. Look for an **Install** icon in the address bar.
   - If you don’t see it:
     - **Chrome:** Menu (3 dots) -> **Cast, save, and share** -> **Install page as app**
     - **Edge:** Menu (3 dots) -> **Apps** -> **Install this site as an app**
3. Confirm install.
4. Open the app from your desktop/start menu.

### Android (Chrome)
1. Open the website URL in **Chrome**.
2. Tap the menu (3 dots).
3. Tap **Install app** (or **Add to Home screen**).
4. Open it from your home screen.

### iPhone / iPad (Safari)
1. Open the website URL in **Safari** (not Chrome).
2. Tap the **Share** button.
3. Tap **Add to Home Screen**.
4. Open it from your home screen.

## 2) Sign In / Roles

### Signing in
1. Open the app.
2. Sign in using your league-provided account.

### Roles
- **Admin**: manages league settings, roles, courses, schedule tools, brackets, etc.
- **Scorer**: enters scores, prints scorecards; may have additional league tools depending on setup.
- **Player**: views schedule/leaderboard, checks payments, participates in league chat.

---

## 3) Player Guide

### Dashboard
- View key league info like announcements, weather, and next match.

### Leaderboard
- View standings and filter by week/date range.

### Schedule
- View weekly matches and events.

### Season Overview
- Read league rules and season info.

### Payments
- View your dues status:
  - **Total Due**
  - **Paid**
  - **Balance**
- Review payment history.

### Shot Tracking (Test Feature)
- The app may show **Premium Required**.
- If your league enabled it, you can log shots (club, distance, lie, result, note).

### League Chat
- Messages are visible to **everyone in your active league**.
- Use chat for quick communication (tee time notes, reminders, etc.).

---

## 4) Scorer Guide

### Enter Scores
1. Go to **Enter Scores**.
2. Select:
   - Week
   - Date
   - Course
   - Tee
   - Player
3. Enter hole-by-hole scores.
4. Save/submit as instructed by league rules.

### Print Scorecards
1. Go to **Print Scorecards**.
2. Select the week.
3. Print or save as PDF for distribution.

---

## 5) Admin Guide

### User Roles
1. Go to **Admin -> User Roles**.
2. Assign roles (Admin / Scorer / Player).

### League Settings
- Configure league options and season settings.

### Courses
- Add/edit course and tee information.

### Schedule / Rainouts
- Use the schedule and rainout tools to manage changes and communicate updates.

### Pairings (Admin)
1. Go to the Schedule page and locate **Pairings (Admin)**.
2. Choose the week.
3. Use one of:
   - **Auto Generate** (deterministic)
   - **Suggest (AI Scaffold)** (balanced suggestion; you can edit before saving)
4. Click **Save Pairings**.

### Brackets (Admin)
1. Go to **Admin -> Brackets**.
2. Generate from players or load an existing bracket.
3. Save the bracket.

### Payments (Admin)
- Configure payment categories/dues.
- Record payments (cash/check/other).
- Send reminders as needed.

### Announcements
- Use Dashboard announcements to post league-wide messages.
- Pin important messages.

---

## 6) Troubleshooting

### I don’t see the latest version / pages look wrong
1. Refresh the app.
2. If using a browser tab:
   - **Windows:** Ctrl+F5
   - **Mac:** Cmd+Shift+R
3. If installed as an app:
   - Close and reopen the app.
   - If still stale, uninstall the app and reinstall from the website.

### I’m seeing someone else’s league chat/data
- Verify you selected the correct **league** (if your league uses multiple leagues).
